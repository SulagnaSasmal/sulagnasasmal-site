Automation
PPT → MP4 Documentation Automation
Converts PowerPoint presentations into narrated MP4 videos — fully automated
An end-to-end automation pipeline that takes a PowerPoint file and produces a narrated MP4 video — no manual recording, no screen capture. The pipeline extracts slide notes, generates AI voice-over using Azure Text-to-Speech (Jenny Neural), sets slide timings via PowerPoint COM automation, and muxes everything into a final video using FFmpeg. Built with FastAPI and a full web UI for upload, job tracking, and download.
PPT → Azure TTS → PowerPoint COM → FFmpeg → MP4
